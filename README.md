## Project Overview

This Netflix Clone replicates the UI design of Netflix, including the homepage layout, banners, navigation bar, and movie sections. It's built entirely with HTML5 and CSS3, making it a great learning project for understanding responsive web design and modern CSS techniques.

## Features ✨

- ✅ Netflix-like homepage UI with modern design
- ✅ Fully responsive design (works on mobile, tablet & desktop)
- ✅ Navigation bar with logo and menu
- ✅ Hero section with full-width background image
- ✅ Movie/series rows layout with smooth scrolling
- ✅ Interactive hover effects on cards and buttons
- ✅ Footer section with links
- ✅ Clean and maintainable code structure

## How I Built This Project

### 1. Basic Structure (HTML)
- Created the main structure using semantic HTML5 elements
- Organized sections:
  - Navbar (with logo and navigation links)
  - Hero Section (eye-catching banner)
  - Movie Rows (multiple sections for different genres)
  - Footer (links and information)

### 2. Styling (CSS)
- Applied Netflix-inspired color scheme (black, red, white)
- Used custom fonts for typography
- Added background images for visual appeal
- Implemented smooth transitions and animations

### 3. Layout Design
- **Flexbox**: For navbar alignment and row flexibility
- **CSS Grid**: For movie thumbnail layouts
- Maintained proper spacing and alignment throughout

### 4. Responsive Design
- Used media queries to adapt layout for different screen sizes:
  - **Mobile**: < 768px
  - **Tablet**: 768px - 1024px
  - **Desktop**: > 1024px
- Optimized touch interactions for mobile devices

### 5. UI Improvements
- Added hover effects on movie cards
- Styled buttons with active states
- Improved visual hierarchy
- Enhanced user experience with smooth animations

## Installation & Setup

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A code editor (VS Code recommended)

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/bhushanpatil876/Netflix-Clone.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd netflix-clone
   ```

3. **Open in your browser**
   - Double-click `index.html` to open directly, or
   - Use a live server extension in VS Code:
     - Install "Live Server" extension
     - Right-click `index.html` → "Open with Live Server"
