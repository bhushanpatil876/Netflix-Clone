Netflix Clone 🎬
Project Overview

This project is a Netflix Clone built using HTML and CSS. It replicates the UI design of Netflix, including the homepage layout, banners, navigation bar, and movie sections. The goal of this project was to practice frontend development skills like layout design, responsiveness, and styling.

Features
Netflix-like homepage UI
Responsive design (works on mobile & desktop)
Navigation bar with logo and menu
Hero section with background image
Movie/series rows layout
Footer section
Technologies Used
HTML5 – Structure of the website
CSS3 – Styling and layout
Flexbox & Grid – For responsive design
Media Queries – For mobile responsiveness
